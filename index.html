Quelques outils de développement Web

Le développement web avec ces technologies et frameworks constituent un vaste domaine disposant de plusieurs outils.Mais dans la plupart des temps, ceux-ci sont couramment utilisés:

HTML5 Markup

Le HTML(HyperText Markup Language) est le langage de balisage de base utilisé pour structurer le contenu d'une page web. Son principal avantage est sa simplicité et sa compatibilité. Le HTML5 ne compte en effet, qu’une centaine de balises. Le HTML5 est incontournable puisque aujourd’hui tous les sites web sont basés sur ce langage. C'est l'un des outils principaux du développement web permettant de créer la structure informatique d'une page internet.

CSS3 Styling

Le CSS ( Cascading Style Sheets ) est un langage de feuilles de style utilisé pour définir la présentation et l'apparence d'un site web codées en HTML ou en XML, en contrôlant les couleurs, les politiques, les marges, etc. Généralement il permet de créer des pages web à l'apparence soignée.

Graphic Design

Le graphisme représente une grande valeur pour les entreprises, car il génère des attitudes en fonction des perceptions et des messages qu'il transmet, mais le plus important est qu'il contribue à créer une idée de marque, une personnalité face aux clients, aux concurrents et au grand public, il cré la fidélité.


Bootstrap

Bootstrap est un framework CSS de Twitter pensé pour développer des sites avec un design responsive, qui s’adapte à tout type d’écran, et en priorité pour les smartphones. Il fournit des outils avec des styles déjà en place pour des typographies, des boutons, des interfaces de navigation et bien d’autres encore.

Joomla style

Joomla est un CMS (content managment system), ce type d’outil permet la gestion autonome d’un blog ou d’un site web. c'est est un système de gestion de contenu , qui vous permet de créer des sites internet de qualité professionnelle.

WordPress Design

WordPress est un système de gestion de contenu (CMS) gratuit et open source qui permet de créer des sites Internet même sans connaissances en programmation. Les plugins et les thèmes permettent d'étendre les fonctions des sites WordPress et de personnaliser les mises en page selon ses besoins.
